# Gomoku (五子棋) Game Project

## Overview
Gomoku, also known as Five in a Row, is a classic board game played between two players. The objective is to be the first to get five of your stones in a row either horizontally, vertically, or diagonally.

## Features
- Simple and intuitive user interface
- Support for both single and multiplayer modes
- Adjustable board size
- Move history and undo functionality

## Technologies Used
- JavaScript for game logic
- HTML/CSS for rendering the game board
- Optionally, can use frameworks like React for an enhanced experience

## Installation
1. Clone the repository: `git clone https://github.com/WnHao1/wu-zi-qi218.git`
2. Navigate to the project directory.
3. Open `index.html` in your web browser.

## How to Play
1. Players take turns placing their stones on the board.
2. The first player to align five stones wins the game.
